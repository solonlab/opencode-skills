# videocut:剪口播

> 口播视频剪辑助手

## 文件清单

| 文件 | 功能 |
|------|------|
| `SKILL.md` | 入口：流程、触发词、核心规则 |
| `tips/口误识别方法论.md` | 核心：口误识别 + 时间戳对齐 |
| `tips/转录最佳实践.md` | 辅助：FunASR 转录参数 |

## 输入输出

```
输入: 01-xxx-v1.mp4 (原始视频)
      ↓
输出: 01-xxx-v1_transcript.json  转录（含时间戳）
      01-xxx-v1_审查稿.md        审查稿（时间戳驱动）
```

## 核心原则

1. **时间戳驱动**：删除任务直接标注 `(start-end)`
2. **逐token分析**：口误需逐token查时间戳
3. **删前面保后面**：重复时删前面版本
